# 14. MVP and Phased Roadmap

## Phase 0 — Documentation core
- spec
- architecture docs
- types
- risk model
- contribution rules

## Phase 1 — Local runtime MVP
- context schema
- evaluator
- component registry
- orchestrator
- trace
- simulator CLI
- one example app

## Phase 2 — Safety maturity
- policy gate
- abstention logic
- safe fallback model
- trust-sensitive metrics
- reviewed learning hooks

## Phase 3 — Commercial readiness
- hosted control plane
- remote policy management
- enterprise logging
- approval workflow
- premium packs

## Phase 4 — Agentic OS coexistence
- external agent hints
- app-native handoff protocol
- context provider APIs
- OS-agent compatibility experiments

## MVP warning
Do not begin with:
- full AI reranking
- marketplace
- multi-tenant dashboards
- deep legacy retrofits

## Build order
1. schema
2. evaluator
3. registry
4. orchestrator
5. trace
6. simulator
7. policy gate
8. examples
